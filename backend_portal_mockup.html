Staff Portal

All tools for programme staff to set up MSMEs from a desktop — before handing the POS device to the trader

ℹ️ Pre-loading products, stock, and stores from the web portal means the trader's POS is ready to use on day one. Much faster than entering everything on a Sunmi 5.99" screen.
📦
Product Master
Create and manage product catalogues per MSME. CSV bulk import with validation.
  • Create / edit products per MSME
  • SKU, barcode, price, cost, tax flag
  • Product category & unit of measure
  • Expiry date (pharmaceutical sector)
  • CSV bulk import with error report
  • Download product template CSV
  • Deactivate / archive products

Stores & Warehouses
Create named store locations, set default POS store, manage stock and transfers.
  • Create named stores per MSME
  • Set default store for POS
  • View stock levels per store
  • Record inter-store transfers
  • Mark stores active / inactive

Opening Stock Setup
CSV bulk import of opening stock per MSME. Manual entry and error reporting.
  • CSV bulk opening stock import
  • Manual single-product entry
  • Assign opening stock to specific store
  • Download stock template CSV
  • View import history & errors

Per-MSME step tracker from registration to first live sale.
